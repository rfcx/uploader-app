# RFCx Uploader
Desktop application for ingesting audio to RFCx platform

## Getting started

1. Install dependencies
    ```
    yarn
    ```

2. Compile and hot-reload for development
    ```
    yarn electron:serve
    ```

## Development 

- Lint and fix files
    ```
    yarn lint
    ```

## Production

- Build for ???
    ```
    yarn electron:build
    ```
